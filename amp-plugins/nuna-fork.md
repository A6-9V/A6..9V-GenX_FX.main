# NUNA Fork Plugin

## Overview
Integration with the NUNA Fork repository for MQL5 Trading Robots and Data Management.

## Repository
URL: https://forge.mql5.io/A6-9V/NUNA_FORK.git

## Components
- **Expert Advisors**: MQ5 files for automated trading.
- **Trading Data Manager**: Python tool for managing trading logs and reports.
- **Automated Deployment**: Gitea Action for VPS deployment.

## Usage
To run the data manager:
```bash
cd nuna_fork
python3 trading_data_manager.py run --apply
```
